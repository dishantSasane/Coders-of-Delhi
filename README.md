# Coders of Delhi Internship Project

This repository showcases my work during the **Coders of Delhi Virtual Internship** with **CWH Solutions**, where I was aksed to apply **pure Python skills** without relying on external libraries 
like `pandas` or `numpy`. The project demonstrates how strong fundamentals in Python can be leveraged to solve real-world problems efficiently and cleanly.

## Project Overview
- **Data Cleaning:** Transformed raw, unstructured company app data into a clean, usable format using only core Python.
- **Mutuals Logic:** Implemented an algorithm to identify mutual connections between users, forming the backbone of social networking features.
- **Pages You Might Like:** Designed a recommendation system that suggests relevant pages to users based on their activity and interests.

## Tech Stack
- **Language:** Python (core only)
- **Focus Areas:** Data structures, algorithms, and clean code practices

## Key Takeaways
- Demonstrates problem-solving without external libraries.
- Highlights efficiency and scalability through clean, modular logic.
- Reflects discipline, creativity, and technical depth in Python programming.
